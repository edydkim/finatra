Announcing the release of Finatra 2.7.0

posted by cacoco on 2017-01-13 10:06:54 -0800

Finatra release 2.7.0 is available from Maven Central.

This release was done over the holidays. A big thanks to @jcrossley!

Some of the changes in this release include:

  • Support for Scala’s scala.concurrent.Future in the CallbackConverter. Now controller callbacks with a return type of scala.concurrent.Future will be converted via a bijection to Twitter’s c.t.util.Future. Thanks to @virusdave for the contribution.

  • Fix for ResponseBuilder.fileOrIndex to verify the requested file actually exists before returning the index. Thanks to @AlexITC for the contribution.

  • Support for Request forwarding with Controllers. Requests can be now “forwarded” from one Controller route to another. See the User’s Guide for more information.

  • The grizzled-slf4j and codingwell/scala-guice dependency versions were updated.


Please checkout the Finatra 2.7.0 release notes on github for more information.

See the Finatra Github page or checkout our User’s Guide for more information on getting started.
