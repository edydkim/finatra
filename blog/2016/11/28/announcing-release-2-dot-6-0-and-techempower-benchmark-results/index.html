Announcing Finatra release 2.6.0 and TechEmpower Round 13 Results!

posted by cacoco on 2016-11-28 09:42:25 -0800

Finatra release 2.6.0 is now available from Maven Central.

Some changes from the latest release (there are several dependency updates as we start to prepare for supporting a Scala 2.12 release):

  • We’ve upgraded to Jackson 2.8.3 (from Jackson 2.6.5) and then to Jackson 2.8.4!
  • We’ve upgraded to Scalatest 3.0.0 and Scalacheck 1.13.4.
  • This site’s source is no longer maitained in the gh-pages-source branch in Github but is now co-located with the code source.
  • Support for “extended boolean” @QueryParam annotated case class fields (e.g., support for “t”, “f”, “1”, “0” parsed as booleans).
  • Dropped the need to add @Inject on the finagle-http Request field in a custom case class request. The finagle-http Request was not actually “injected” and use of the @Inject annotation as a marker caused undue confusion. See the documentation here for more information.

Note: Please update your dependencies accordingly.

Please checkout the finatra-2.6.0 release notes on github for more information.
